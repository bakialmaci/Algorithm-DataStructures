How to compile 32-bit program on 64-bit gcc in C and C++

Command: gcc -v -> too see what bit system.

Command: gcc -m32 source.c -o source

32-Bit compilation
    Input: gcc -m32 geek.c -o out
    Output: ./out
    Size = 4
